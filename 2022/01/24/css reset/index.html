Reset css or Normalize css

 |  2022-01-24

CSS 스타일링을 하기 전

각 브라우저에는 user agent style, 즉 각 브라우저들만의 스타일 기본값이 있다. 개발을 하다보면 이 user agent style 때문에 css를 적용할 때 브라우저마다 다른 결과물을 보여주게 되는데, 최대한 같은 결과물을 보여주기 위해 CSS 초기화를 사용하게 된다.

CSS 초기화를 하는 두가지 방법이 있다.

  1. Reset css
  2. Normalize css

reset.css 파일을 만들어 자신의 입맛으로 기본값을 설정해도 되고 Reset css 를 검색해 대중적인 코드를 복사해 사용해도 된다. 가져온 코드에 자신의 스타일을 더해 사용하는 방법도 있다. 요즘에는 Reset css 대신 Normalize css도 많이 사용한다고 한다. 두 방법 모두 코드를 저장한 파일을 head 안에 link로 불러오면 된다.

Reset CSS 적용

Reset css는 모든 요소의 스타일을 0으로 만든 상태에서 디자인을 해나갈 수 있습니다. 카드의 마진과 카드안 텍스트까지 공간이 사라졌다.

끝으로

이제까지 CSS를 사용하면서 요소의 패딩이며 마진이며 성실하게 0으로 기본 값을 주곤 했는데.. Reset css와 Normalize css를 상황에 맞게 사용해 개발하면 좋을 것 같다. 개인 프로젝트를 할 때에는 Reset css가 더 마음에 갈 것 같다.
