Axios

 |  2022-02-07

Axios 란?

Axios는 node.js와 브라우저를 위한 Promise 기반 HTTP 클라이언트입니다.

Axios 특징

  • 브라우저를 위해 XMLHttpRequests 생성
  • node.js를 위해 http 요청 생성
  • Promise API를 지원
  • 요청 및 응답 인터셉트
  • 요청 및 응답 데이터 변환
  • 요청 취소
  • JSON 데이터 자동 변환
  • XSRF를 막기위한 클라이언트 사이드 지원

Fetch와 다른점?

  • fetch는 옛날 브라우저에는 지원이 안되는 반면에 Axios는 옛날 브라우저까지 지원을 해줘 호환성이 높다.
  • fetch를 사용하면 JSON 데이터를 변환을 직접 해줘야 하고, Axios를 활용하면 JSON 데이터 자동 변환되어 결과값을 도출한다.
  • Fetch는 body 속성을 사용하고, Axios는 data 속성을 사용한다.
  • Fetch는 응답객체가 ok 속성을 포함하면 성공이고, Axios는 status가 200이고 statusText가 ‘OK’이면 성공이다.
  • Axios는 HTTP 요청을 가로챌 수 있고 요청을 취소할 수 있지만 Fetch는 그러한 기능이 없다.

Axios 시작하기

npm을 이용하고 싶다면 Axios 공식 페이지를 확인

  1. 터미널에 yarn add axios 를 입력해 라이브러리 추가

  2. package.json에서 Axios가 설치되었는지 확인

    package.json

  3. Axios를 사용할 페이지에 연결하기

    import axios from 'axios

  4. Axios 인스턴트 만들기

    axios.create({config} : 사용자 지정 config로 새로운 Axios 인스턴트를 만들 수 있다.

  5. 인스턴트 메소드

    axios#get(url[,config])
