단축평가

 |  2022-02-06

논리 연산자를 사용한 단축평가

&&(논리곱연산자)는 두 개의 피연산자가 모두 true일 때 두번째 피연산자를 반환한다. &&는 좌항에서 우항으로 평가되므로 좌항의 피연산자가 Truthy값일 경우 우항으로 넘어가 우항의 피연산자를 평가하게 된다.

||(논리합연산자)는 두 개의 피연산자 중 하나만 true로 평가된다면 true를 반환한다. 첫번째 피연산자가 Truthy값일 경우 우항의 평가를 하지 않고 좌항의 피연산자를 반환하게 된다.


단축평가란

이처럼 논리 연산의 결과를 결정하는 피연산자를 타입 변환하지 않고 그대로 반환하는 것을 단축평가라고 한다. 표현식을 평가하는 도중에 평가 결과가 확정된 경우 나머지 평과과정을 생략한다.

단축평가로 대체 가능한 것들

  • 조건이 Truthy인 값으로 무언가를 해야 할 때 if문을 &&을 활용한 단축평가로 대체할 수 있다.
  • 조건이 Falsy인 값으로 무언가를 해야할 때 if문을 ||을 활용한 단축평가로 대체할 수 있다.

단축평가를 유용하게 사용하기

  • 객체를 가르키기를 기대하는 변수가 null 또는 undefined가 아닌지 확인하고 프로퍼티를 참조하고 싶을 때
  • 함수 매개변수에 기본값을 설정할 때

옵셔널 체이닝 연산자

ES11(ECMAScript2020)에서 도입된 옵셔널 체이닝 연산자 ?.는 좌항의 피연산자가 null 또는 undefined일 경우 undefined를 반환하고, 그렇지 않을 경우 우항의 프로퍼티 참조를 이어간다.

null 병합 연산자

ES11(ECMAScript2020)에 도입된 null 병합 연산자 ??는 좌항의 피연산자가 null 또는 undefined일 경우 우항의 피연산자를 반환하고, 그렇지 않으면 좌항의 피연산자를 반환한다.

null 병합 연산자는 변수에 기본값을 설정할 때 유용하다. null 병합 연산자가 도입되기 이전에는 || 을 사용했는데, ||는 첫번째 피연산자가 Falsy값이라면 우항의 피연산자를 반환하는데 변수 설정시 기본값으로 0이나 ‘ ‘ 이 유효한 변수를 설정하려면 원하지 않는 값이 나올 수 있다.

null 병합 연산자는 Falsy값이라도 null이나 undefined가 아니라면 좌항의 피연산자를 그대로 반환한다.
